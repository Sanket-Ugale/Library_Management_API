Library Management API

API for managing library books and users

API Endpoints

  • /api/books - Get all books
  • /api/books/1 - Get book by ID
  • /api/books/1/users - Get users who have borrowed book by ID
  • /api/users - Get all users
  • /api/users/1 - Get user by ID
  • /api/users/1/books - Get books borrowed by user by ID
